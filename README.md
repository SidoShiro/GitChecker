# Git Checker

# Usage

* Select a specified dir
```sh
./gitChecker -root [DIR] 
```
* Auto local dir
```sh
gitChecker
```
* Show details
```sh
gitChecker -verbose
```

* Help
```sh
gitChecker -help
```

# Install

```sh
go build gitChecker
```

## Personal install

> Move gitChecker binary in `/home/sido/go/bin/` 

* `mv gitChecker /home/sido/go/bin/gitChecker`
